# Module Boundaries

## Modules

### Business modules (`internal/<module>/`)

- `accesscontrol/` — owns: AuthUser, AuthContext, Policy, Permissions, UnauthorizedRoleError, UnknownActionError. Depends on: (none — stdlib only)
- `catalog/` — owns: Book, Copy, BookId, CopyId, Isbn, BookCondition, CopyStatus, BookNotFoundError, CopyNotFoundError, DuplicateIsbnError. Depends on: accesscontrol, shared/isbngateway, shared/bookcache, shared/filestorage, shared/tx, shared/events, shared/db, shared/http
- `membership/` — owns: Member, MemberId, MembershipTier, MembershipStatus, MemberNotFoundError, DuplicateEmailError. Depends on: accesscontrol, shared/tx, shared/events, shared/db, shared/http
- `lending/` — owns: Loan, LoanId, Reservation, ReservationId, LoanOpened, LoanReturned, ReservationQueued, ReservationFulfilled, ReservationUnfulfilled, AutoLoanOpened, AutoLoanFailed, CopyUnavailableError, LoanNotFoundError, MemberIneligibleError. Depends on: accesscontrol, catalog, membership, shared/events, shared/tx, shared/db, shared/http
- `fines/` — owns: Fine, FineId, FineAssessed, MemberAutoSuspended, FineNotFoundError, FineAlreadyPaidError. Depends on: accesscontrol, lending (facade only), membership (facade only), shared/events, shared/tx, shared/db, shared/http
- `categories/` — owns: Category, CategoryId, CategoryNotFoundError. Depends on: accesscontrol, shared/tx, shared/db, shared/http
- `chat/` — owns: Message, MessageId, ConversationId, ChatFrame, ChatDelta, MessageNotFoundError. Depends on: shared/chatgateway, shared/http

### Shared infrastructure (`internal/shared/<package>/`)

- `shared/db/` — owns: bun client factory, migration runner. Depends on: stdlib + third-party only.
- `shared/events/` — owns: EventBus interface, InMemoryEventBus, DomainEvent interface, Unsubscribe. Depends on: stdlib only.
- `shared/http/` — owns: DomainErrorMiddleware, WriteJSON helper, ErrorResponse, request-scoped helpers. Depends on: stdlib + chi only.
- `shared/tx/` — owns: TransactionalContext interface, InMemoryTransactionalContext, BunTransactionalContext. Depends on: shared/events, shared/db, stdlib.
- `shared/isbngateway/` — owns: IsbnLookupGateway port + impls (in-memory, external). Depends on: stdlib + third-party only.
- `shared/bookcache/` — owns: BookCacheGateway port + impls (in-memory, Redis). Depends on: stdlib + third-party only.
- `shared/chatgateway/` — owns: ChatGateway port + impls (in-memory, OpenAI). Depends on: stdlib + third-party only.
- `shared/filestorage/` — owns: FileStorageGateway port + impls (in-memory, ...). Depends on: stdlib + third-party only.

### Composition root

- `cmd/library/` — owns: main.go, wires everything. Depends on: every business module + shared/*.

## Rules

- **Modules may only import from declared dependencies.** Imports outside this list are violations.
- **Cross-module deps go facade-to-facade.** Never reach into another module's repository, types, or HTTP layer.
- **No cross-module DB joins.** Bun queries inside a module touch only that module's tables. Cross-module reads use the other module's facade (with batch methods to avoid N+1).
- **No cross-module transactions.** A `TransactionalContext` instance is created and used inside ONE module.
- **No shared internal types between modules.** Canonical type lives in its owning module; consumers import it.
- **HTTP DTOs never escape `<module>/http/`.** Facade methods take and return domain DTOs only.
- **No mocks.** Test substitution uses in-memory implementations; fault injection uses unexported spec-local decorator wrappers.
- **Shared infrastructure NEVER imports business modules.** The dependency direction is strict: `cmd → business modules → shared`.
- **No circular dependencies.** When in doubt, duplicate code rather than create a coupling that violates boundaries.
- **No `init()` for module wiring.** All wiring is explicit in `cmd/library/main.go`.
